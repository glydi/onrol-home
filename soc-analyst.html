Why This Program Exists

Companies don't need more certificate-holders. They need analysts who can actually do the work.

This program trains you to cut through alert noise, investigate real threats and respond effectively — using the same industry tools and frameworks a working SOC runs on every day.

Why ONROL

ONROL is a hands-on execution school that skips pure theory. You'll train like a working security analyst — investigating real incidents using tools like Splunk, ELK, Wireshark, YARA and live event logs.

Less watching. More doing. You leave shift-ready.

What You'll Learn

A 12-module curriculum, grouped into 5 progressive segments that mirror how a real SOC operates.

Segment 01 — Foundations

Computer Fundamentals + Networking Basics

Operating systems, CLI, virtualization, the OSI/TCP-IP stack, IP addressing, ports, protocols, troubleshooting and lab setup.

OutcomeThe technical baseline every analyst needs before touching a SIEM.

Segment 02 — Packets, Logging & SIEM

Packets & Logging + SIEM Basics

Packet capture, HTTP/DNS analysis, Windows and Linux logs, log forwarding and parsing, SIEM architecture, Splunk SPL, Kibana, dashboards and alerts.

OutcomeIngest, search and visualize security data in a working SIEM.

Segment 03 — Threat Detection & IDS

Threat Detection + Packet Analysis & IDS

Brute-force, port-scan, malware and phishing indicators, web-attack detection, advanced Wireshark, Zeek, NetFlow and Suricata signatures.

OutcomeDetect real attacks in real traffic and logs.

Segment 04 — Endpoint & Incident Response

Endpoint Security & Investigation + Incident Response

EDR, persistence, memory acquisition and analysis, YARA, the NIST IR lifecycle, triage, containment, evidence and chain of custody, plus tabletop exercises.

OutcomeInvestigate an endpoint compromise and run a clean incident response.

Segment 05 — Hunting, Advanced SIEM/SOAR & Career

Threat Intel & Hunting + Advanced SIEM & SOAR + Capstone + Placement Prep

IOC management, hunt methodology, UEBA, correlation rules, SOAR playbooks, a full incident simulation, and resume & interview preparation.

OutcomeHunt proactively, automate response and walk in interview-ready.
